# AI in Cambodia Website

This project is focused on showcasing the advancements and initiatives in artificial intelligence within Cambodia. It serves as a platform for sharing knowledge, resources, and updates related to AI in the region.

## Project Structure

- `src/css/style.css`: Custom styles for the website.
- `src/css/bootstrap.min.css`: Minified Bootstrap CSS framework.
- `src/js/main.js`: JavaScript for interactivity and functionality.
- `src/pages/index.html`: Main entry point of the website.
- `public/assets`: Directory for static assets like images and fonts.
- `.gitignore`: Specifies files to be ignored by Git.
- `vercel.json`: Configuration for deploying on Vercel.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd ai-cambodia-website
   ```

3. Open the `index.html` file in your browser to view the website.
